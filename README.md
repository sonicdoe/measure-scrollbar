# measure-scrollbar

> Get the width of the browser scrollbar

## Installation

```
$ npm install measure-scrollbar
```

## Usage

```js
import measureScrollbar from 'measure-scrollbar'

measureScrollbar()
// => 15
```

`measureScrollbar()` returns the width of the browser scrollbar in pixels.
